Knowledge Manager

Render your Obsidian vault as a browsable, print-friendly website. Mac-native app, one click to start.

Download

Pick the build that matches your Mac. Requires macOS 11 or later.

Apple Silicon
M1 / M2 / M3 / M4
Download arm64 →
Intel Mac
2019 & earlier models
Download x64 →

Not sure which one? Click the Apple menu → About This Mac. If it says "Chip: Apple M…", pick Apple Silicon. Otherwise pick Intel.

Install

  1. Unzip the download.
  2. Drag Knowledge Manager.app into your Applications folder.
  3. Right-click the app → Open. (Needed once. The app isn't code-signed yet, so macOS needs permission.)
  4. Click Open on the Gatekeeper dialog.

First launch

  1. Choose your Obsidian vault folder when prompted.
  2. Dependencies install (one-time, ~2 min, needs internet).
  3. Browser opens automatically at http://localhost:8080.

Daily use

Start
Double-click Knowledge Manager.app.
See it's running
The app appears in the Dock while the server is active.
Stop
Right-click the Dock icon → Quit, or ⌘Q while focused.
Re-open browser
Double-click the app while it's running — you'll get an Open Browser / Stop dialog.

Troubleshooting

"Cannot verify the developer"

Right-click the app → Open (not double-click) on first launch. This bypasses Gatekeeper for unsigned apps.

Server fails to start

Check the log at ~/Library/Application Support/Knowledge Manager/server.log.

Vault folder moved

Delete ~/Library/Application Support/Knowledge Manager/vault.path and relaunch. You'll be asked to pick a new vault.

Reset everything

Remove ~/Library/Application Support/Knowledge Manager/ and relaunch. All per-user state lives there.
